#  Glowrora — Frontend Project

Glowrora è un progetto frontend statico realizzato con HTML e CSS e js di base .
Il sito rappresenta un brand moda femminile con un design moderno, elegante e responsive.


##  Obiettivo del progetto

Realizzare un sito web multipagina applicando:

- HTML semantico
- CSS Grid
- Flexbox
- Responsive Design
- Transizioni
- Animazioni CSS
- Struttura organizzata e scalabile
- javascript base 

Il progetto è stato sviluppato come lavoro da portfolio.



##  Tecnologie utilizzate

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Responsive Media Queries
- Animazioni e transizioni CSS



## Caratteristiche principali

✔ Layout completamente responsive  
✔ Hero section con overlay  
✔ Grid prodotti moderna  
✔ Card con hover effects  
✔ Animazioni di entrata  
✔ Design coerente su tutte le pagine  
✔ Struttura pronta per evoluzione JavaScript  


## Responsive Design

Il sito è ottimizzato per:

- Desktop
- Tablet
- Mobile

Le griglie si adattano dinamicamente alla dimensione dello schermo.



##  Evoluzione futura

Il progetto è strutturato per permettere:

- Aggiunta di funzionalità JavaScript
- Gestione carrello
- Manipolazione DOM
- Interazioni dinamiche
